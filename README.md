### 주요 화면 소개
- 메인화면
  - 상단의 메뉴는 고정 된 채 마우스 Hover시 밑줄이 그어집니다.
![image](https://github.com/leesoyuun/Portfolio/assets/51051548/97619a20-fa1c-48e2-82c1-9372bd510d20)

- About Me
   - 추후 추가될 내용을 위하여 JSON을 사용하여 이력 / 수상내역 / 기술발표내역 등을 저장하였습니다.
  ![image](https://github.com/leesoyuun/Portfolio/assets/51051548/944a1ff0-74a4-409b-9f51-51e1d56941aa)
  
 
- PROJECT
  - 프로젝트 내용 또한 JSON을 사용하여 정리하였습니다.
  - CODE클릭시 해당 깃허브로 들어가지며 Live Demo 클릭시 배포된 페이지로 넘어가게 됩니다.
  - 또한, 논문페이지를 클릭하게 되면 DBpia에 업로드된 논문을 볼 수 있습니다.
  - 해당 프로젝트 명 / 사진 / MoreDetail을 클릭할시 자세한 프로젝트 소개페이지로 넘어가게 됩니다.
  - 프로젝트명에 마우스를 올릴시 자동으로 밑줄이 그어지는 애니메이션을 구현하였습니다.
    <br>
  ![image](https://github.com/leesoyuun/Portfolio/assets/51051548/e49e0804-a2af-4d6b-9353-0305c9daa960)

- Detail Project
  - 프로젝트의 디테일한 정보를 볼 수 있는 페이지입니다.
  - React의 UseParams()를 사용하여 url에 따라 값이 바뀌도록 지정하였습니다.
  - 디테일 프로젝트를 들어갈때 자동적으로 스크롤이 맨 상단으로 올라갈 수 있도록 구현하였습니다.
 
    ![image](https://github.com/leesoyuun/Portfolio/assets/51051548/bf9a3f12-fcd4-4d54-bbea-98112f23623a)


- Tech Skill
  - Lottie를 사용하여 Gif를 구현하였습니다.
  - 나뉘어진 각 분야의 테크스길을 클릭할 시 보유하고 있는 테크스길이 나오며 그래프가 채워지는 애니메이션을 라이브러리 없이 직접 구현하였습니다.
  - 해당 정보도 전부 JSON으로 저장하여 불러오는 형식을 사용하였습니다.
   ![image](https://github.com/leesoyuun/Portfolio/assets/51051548/94e814cb-6bc1-4a8c-9b95-bae414a0822b)
